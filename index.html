  1. 对于推送出现权限不足情况需要去git hub 配置ssh秘钥。
    1. ssh-keygen -t ed25519 -C "you github email"创建密钥
    2. ssh-add ~/.ssh/id_ed25519 添加密钥
    3. cat ~/.ssh/id_ed25519.pub 获取到当前密钥
    4. 在Github 的Setting -》SSH and GPG keys里新增一个SSH密钥，然后将 c 获取的密钥粘贴并保存
    5. ssh -T git@github.com 显示成功即配置完成
